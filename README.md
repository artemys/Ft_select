# Ft_select
2eme projet de la branche UNIX pour l'école 42.
- Interface utilisateur via le terminal.
- Bibliothèque Termcaps.

Usage : 

Make 


./ft_select choix1 choix 2 choix 3

Utilisez les touche flecher pour ce deplacer dans la liste.

esc pour quitter.
